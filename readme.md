# ImageArchive

## 归档流程

### 选择工作目录

您当前存放图片的文件夹

### 选择归档目录

归档后的文件夹，建议选择空的文件夹

### 相关设置选项

- 命名格式
- 去重
- .......

### 开始归档

1. 遍历工作目录
2. 筛选图片文件
3. 文件预处理
    - 是否快捷方式
    - 是否已归档
    - 是否超出大小
    - 尝试得出最早创建时间
    - 文件后缀
    - 文件md5
    - 文件标签
4. 文件处理

#### 文件处理

1. 复制文件
2. ~~更新JSON数据库 YYYYMM.json 以及 count.json~~
3. 是否对源文件改名
4. 是否删除源文件
5. 是否生成快捷方式
6. 更新 sqlite 数据库

### 监听记录工作目录下的文件列表

这个对于生成快捷方式搭配使用，通过对快捷方式进行归类，实现对已归档的图片进行同步更新

## 恢复流程

## 使用建议

1. 将需要进行归档的图片都放到一个文件夹下（工作目录）
2. 选择一个空的文件夹 (甚至是一个磁盘) 作为归档目录
3. 归档时，选择删除源文件，并创建快捷方式
4. 如此，归档完成后，已归档的源文件都会被替换成快捷方式
5. 在工作目录下，对快捷方式重新进行分类整理即可
